Місія

Будувати надійні інструменти, які допомагають запускати і підтримувати власну інфраструктуру без зайвої складності.

Що я роблю зараз

  • Розвиваю Yunexal Panel як платформу керування серверами на Docker.
  • Працюю над швидкою та стабільною backend-архітектурою на Rust + Axum.
  • Спрощую досвід для self-hosted адміністрування.

Флагманський проєкт

Yunexal Panel

GitHub репозиторій

Self-hosted server management platform built on Docker.

  • Built with Rust + Axum, SQLite, and Bollard (Docker SDK).
  • Templates and static assets are compiled into a single binary.
  • No external runtime files are required.

FAQ

Що таке Yunexal?

Yunexal - неофіційна організація, де я розвиваю власні інфраструктурні продукти та експерименти.
